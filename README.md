Task1:Big Data Analysis

📊 Amazon Product Reviews Analysis using PySpark

 📌 Project Overview

This project performs **big data analysis on Amazon product reviews** using **PySpark**. It demonstrates how to process large datasets efficiently and extract meaningful insights such as review patterns, ratings distribution, and trends.

🚀 Features

* Load and process large CSV datasets using PySpark
* Data cleaning and preprocessing
* Analyze product reviews and ratings
* Generate insights from big data
* Scalable data processing using Spark

🛠️ Technologies Used

* Python 🐍
* PySpark 🔥
* Apache Spark
* Jupyter Notebook


📊 Workflow

1. Create Spark Session
2. Load Dataset
3. Data Cleaning
4. Data Analysis
5. Insights Extraction

 📌 Key Learning Outcomes

* Understanding PySpark basics
* Working with large datasets
* Performing distributed data processing
* Extracting insights from real-world data

Dataset
The dataset used:
Amazon_Product_Reviews.csv
